This folder contains the data and code allowing to reproduce the results presented in Figures 1f, 2e, 2f, 3f, 3g, 4a, 4b, and 4c of the manuscript "Transcription Factor Switching Drives Subtype Specific Pancreatic Cancer" of Rao S. et al (2026, Nature Genetics).

the folder data/ contains R data files (extension ".rdata", to be loaded in R with the function base::load) related to each Figure.

the folder code/ contains R scripts (extension ".r") related to each Figure. Each script loads relevant rdata files, attaches required R packages and then performs the analyses.
